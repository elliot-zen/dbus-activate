# Dbus-activate

Activate application that is on the tray;

## Use

**List all  of the application ids that are in the tray**

```sh
dbus-activate --list 
```

**Activate Application by id**

```sh
dbus-activate <applicationId>
// ex: dbus-activate wechat
```
